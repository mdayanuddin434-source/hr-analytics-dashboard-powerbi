# HR Analytics Dashboard (Power BI Project)

## 📊 Project Overview
Developed an HR analytics dashboard using Power BI to analyze employee attrition, workforce trends, and job satisfaction metrics.

## 🛠 Tools Used
- Power BI (DAX, Dashboard)
- Excel (Data Cleaning)

## 🔍 Key Insights
- Attrition rate is highest among employees with <2 years of experience (~40%)
- Low job satisfaction is a key driver of employee exits
- Certain departments show higher attrition compared to others

## 📈 Outcome
Provided actionable insights to improve employee retention strategies and workforce planning.
